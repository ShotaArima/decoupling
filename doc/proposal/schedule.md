# Schedule: 2026-07-31 に向けた論文・発表計画

更新日: 2026-06-30

## 現在地

本日 2026-06-30 時点で、新しい大規模実験を増やす段階はいったん終える。
以降の主作業は、6 月 30 日のゼミ資料とコメントを踏まえ、研究の主張を原稿として通る形に固定することである。

特に優先する修正は次の 4 点である。

1. 売上 $y$ そのものではなく、基準値からの残差 $r=y-b$ を扱う理由を明確にする。
2. global/local から series/day/hour/interaction へ進む論理を、小売需要の文脈で自然に説明する。
3. latent を分けるだけでは不十分であり、出力された補正量を 4 成分として出す必要があることを明示する。
4. 平均ゼロ制約は予測値を直接良くするためだけでなく、成分の担当範囲を固定するための制約として説明する。

現時点の論文の核は次である。

```text
小売需要では、単純な基準値で説明できる部分が大きい。
本研究では、基準値を置き換えるのではなく、
基準値から外れた残差を series / day / hour / interaction の
出力補正量として分けて出す。
これにより、残差に構造が残る条件では、
予測補正と成分解釈を同時に行える。
```

## 7 月の作業方針

2026-07-31 には、大学の中間発表と国際会議 CfP の締切が重なる。
そのため、発表資料と投稿原稿を別々に作り始めるのではなく、まず日本語論文を先に固める。

作業順は次の通りにする。

```text
日本語論文
  -> 7 分発表用スライド
  -> ゼミで発表練習
  -> 副査 2 名への事前発表
  -> 中間発表版の修正
  -> 英語投稿原稿
  -> 教授確認
  -> CfP 提出
```

この順序にする理由は、発表資料だけを先に作ると、数式、実験の読み方、限界の説明が浅くなりやすいためである。
日本語論文として一度筋を通し、その短縮版として 7 分スライドを作る。
