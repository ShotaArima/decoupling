# Paper Direction: 6月末原稿に向けた主張の再整理

## 1. ゼミコメントから見えた課題

ゼミでの主な指摘は、次の 4 点に整理できる。

| 指摘 | 問題 | 修正方針 |
|---|---|---|
| 定式化と文章の行間が大きい | 数式が何を説明しているのか伝わりにくい | 数式の前に、誰が何を知りたいのかを書く |
| 基準値がなぜ必要か分かりにくい | なぜ $y$ を直接扱わず、$y-b$ を扱うのか不明 | 基準値を「既に説明できる部分」として明示する |
| 直交分解という語が強すぎる | 行列分解や厳密な数理分解を連想させる | 本文では「成分分け」「出力成分モデル」を使う |
| 誰に恩恵があるか不明 | モデルの社会的・実務的な意味が弱い | 小売運用者、発注・補充担当者、分析者を明示する |

## 2. 誰に対して何を作るのか

本研究で想定する利用者は、売上予測モデルそのものを研究する人だけではない。

主な利用者は、次のような小売データを扱う人である。

| 利用者 | 知りたいこと | 提案モデルが返すもの |
|---|---|---|
| 発注・補充担当者 | どの時間帯で基準より外れやすいか | 時間帯成分 $c_{i,h}$ |
| 店舗運営者 | どの店舗・商品が基準からずれやすいか | 系列成分 $g_i$ |
| 販促担当者 | 値引きや休日が特定時間帯に効くか | 日×時間帯成分 $u_{i,d,h}$ |
| 需要分析者 | 基準値で説明できないズレがどこに残るか | 残差成分 $g,a,c,u$ |
| モデル開発者 | 強い baseline の後に学習余地があるか | residual target 診断 |

つまり、提案モデルは「強い予測器をさらに強くする」ことだけを目的にしない。

目的は、基準値から外れた需要を、運用上読める単位に分けることである。

## 3. なぜ $y$ を直接扱わず、$y-b$ を扱うのか

小売需要では、売上 $y$ の大部分が単純な基準値 $b$ で説明できることがある。

例えば、同じ商品の同じ時間帯の直近平均は非常に強い。

このとき、売上を直接

$$
y_{i,d,h}
$$

として学習すると、モデルは既に分かっている大きなパターンを再学習しやすい。

一方で、本研究が知りたいのは、基準値で説明できなかった部分である。

そこで、

$$
r_{i,d,h}
=
y_{i,d,h}
-
b_{i,d,h}
$$

を学習対象にする。

この式は「売上を標準化している」だけではない。

実務的には、次の問いに変換している。

```text
売上はいくらか？
```

ではなく、

```text
既に使える基準値から、どの方向に、どの要因で外れるか？
```

を問うている。

この問いに変えることで、基準値を置き換えるのではなく、基準値の弱い部分を補正するモデルとして位置づけられる。

## 4. 新規性の置き方

従来の local-global 分離では、主に次の 2 種類の表現を考える。

$$
z_g,\quad z_l
$$

小売では local の中に、少なくとも次の複数の意味が混ざる。

| local に混ざる要素 | 例 |
|---|---|
| 日の効果 | 休日、天気、販促日 |
| 時間帯の効果 | 朝、昼、夕方、夜 |
| 日×時間帯の効果 | 休日の夕方、雨の日の昼 |

したがって、本研究では local を 1 つの表現として扱わず、残差出力を次の 4 成分に分ける。

$$
\hat r_{i,d,h}
=
\hat g_i
+ \hat a_{i,d}
+ \hat c_{i,h}
+ \hat u_{i,d,h}
$$

| 成分 | 何を表すか |
|---|---|
| $\hat g_i$ | 店舗・商品系列のズレ |
| $\hat a_{i,d}$ | 日単位のズレ |
| $\hat c_{i,h}$ | 時間帯単位のズレ |
| $\hat u_{i,d,h}$ | 日と時間帯の組み合わせによるズレ |

ここが新規性である。

```text
従来の global/local 分離を、小売需要に合わせて
系列・日・時間帯・日×時間帯の 4 成分へ拡張する。
```

## 5. 「直交分解」という語の扱い

本文では「直交分解」という語を主張の中心に置かない。

理由は、行列分解や厳密な線形代数的分解を連想させ、提案の意図より強く読まれるためである。

本文では、次の表現を使う。

| 避ける表現 | 使う表現 |
|---|---|
| 直交分解 | 成分分け |
| 残差直交分解 | 残差成分モデル |
| 直交制約 | 平均ゼロ制約、主効果を持たない制約 |
| 同定保証 | 出力成分を検証可能にする条件 |

数理的な説明が必要な箇所では、appendix で ANOVA 的な平均分解として説明する。

## 6. 本文での主張案

本文では、次の流れにする。

1. 小売需要では、強い基準値で説明できる部分が大きい。
2. したがって、売上 $y$ を直接説明するより、基準値からのズレ $r=y-b$ を説明する方が実務上意味がある。
3. 従来の global/local 分離では、小売で重要な日・時間帯・日×時間帯の違いが local に混ざる。
4. 提案モデルは、残差出力を系列・日・時間帯・日×時間帯の 4 成分に分ける。
5. synthetic では、真の成分を回復できることを確認する。
6. FreshRetailNet では、時間帯構造が残る residual target で補正と解釈が有効であることを示す。
7. 強い same-hour baseline 後では、残差構造が薄くなり、改善と解釈性が弱くなることを限界として示す。
