## 各ページのコンテンツ案

論文原稿（`doc/conference/apiems2026/paper/contents/*.tex`）の内容に基づく。各ページに「意図」を明記する。

### P1: 表紙

- タイトル、所属、氏名、発表日
- **意図**: タイトルだけで「売上そのものではなく、基準値からのズレを読む研究」だと伝わる表現にする。目次ページは置かないので、口頭で発表の流れを一言添える

### P2: 研究背景（1）— 小売需要予測と強い基準値

- 小売の需要予測は発注・補充・値引き・売場運用の意思決定基盤である
- 食品・日用品では日次合計だけでなく、店舗×商品×日×時間帯の粒度が必要
- 実務では、直近平均・同時間帯平均のような**単純な基準値が強い**
- 高度なモデルに売上全体を再学習させても、基準値で既に説明できる構造を繰り返し学習するだけになりやすい
- **意図**: 「複雑なモデルで精度を上げる」話ではないという前提を最初に共有し、問題の土台（強い基準値の存在）を固定する

### P3: 研究背景（2）— 問いの転換と研究目的

- 担当者が知りたいのは「何個売れるか」だけではない。「基準値からどの方向に・どの単位で外れるか」である
- 例: 夕方だけ基準値を上回る → 補充タイミング見直し。休日の特定時間帯だけズレる → 販促・人員配置
- 研究目的: 残差 $r_{i,d,h}=y_{i,d,h}-b_{i,d,h}$ を解釈可能な成分の和として出力し、**補正と説明を同時に行う**
- 最終予測は $\hat y = b + \hat r$
- **意図**: 「売上はいくらか」→「基準値からどう外れるか」という問いの転換を1枚で見せ、研究目的を1文に固定する

### P4: 関連研究（1）— global/local 表現分離とその限界

- 時系列表現学習では静的/動的、global/local に潜在表現を分ける系譜がある（FHVAE、Disentangled Sequential VAE、Tonekaboni らの decoupling）
- これらの主対象は**潜在空間の分離**であり、出力された予測補正がどの要因に対応するかまでは保証しない
- 小売需要では「local」の中に日・時間帯・日×時間帯が混ざる → global/local の二分は運用上読みたい粒度に対して粗い
- **意図**: 本研究の出発点となる研究系譜を示しつつ、「小売に持ち込むと粒度が粗い」という課題を立てる

### P5: 関連研究（2）— 残差化の系譜と分解型予測

- 残差化: Frisch–Waugh–Lovell、Robinson の部分線形モデル。「説明済みの構造を除いた後に残る変動」を分析対象にする考え方は統計学で確立している
- 残差診断: Breusch–Pagan、Ljung–Box、ARCH。基準モデル後の残差構造を調べることは確立した方法論
- global-local 予測（Deep Factors、DeepGLO）や分解型予測（Autoformer、FEDformer 等）はあるが、分解粒度は trend/season や scale であり、**系列・日・時間帯・相互作用という小売運用単位ではない**
- **意図**: 「残差を対象にする」ことに理論的な裏付けがあることを示し、既存の分解粒度と本研究の粒度の差分を明確にする

### P6: 本研究の位置づけ — 3つの差分

1. 売上 $y$ を直接分解するのではなく、基準値後の残差 $r=y-b$ を主対象にする
2. local を1つの潜在変数として扱わず、系列・日・時間帯・日×時間帯という**運用単位**へ分ける
3. 潜在表現の識別ではなく、**出力される補正成分**に平均ゼロ制約を課す（latent disentanglement ではなく output-level residual decomposition）
- **意図**: 新規性を3点に絞って明示する。質疑「既存研究との違いは？」への先回りにもなる

### P7: 提案手法（1）— モデル全体像

- 全体図: 入力（売上履歴・残差履歴・マスク・日/時間帯特徴）→ Encoder（4表現）→ 成分別 Decoder head → 中心化 → 4成分の和 $\hat r = \hat g + \hat a + \hat c + \hat u$
- 独立に学習した補正モデルの足し合わせではなく、単一損失で end-to-end に同時学習
- **意図**: モデルの全体構造を1枚の図で掴ませる。ここが発表の折り返し地点

### P8: 提案手法（2）— 4成分の意味と分解の非一意性

- $g_i$: 系列固有のズレ／$a_{i,d}$: 日のズレ／$c_{i,h}$: 時間帯のズレ／$u_{i,d,h}$: 日×時間帯のズレ
- 具体例で説明: 常に高く売れる → $g$、特定の日だけ → $a$、夕方だけ → $c$、休日の夕方だけ → $u$
- ただし制約がなければ分け方は**一意ではない**（$g$ に定数を足し $a$ から引いても和は不変）
- **意図**: 「なぜ制約が必要か」を自由度の問題として腹落ちさせる。次ページへの伏線

### P9: 提案手法（3）— 平均ゼロ制約（中心化）

- 日成分は日方向平均0、時間帯成分は時間帯方向平均0、相互作用成分は両方向の主効果を除去
- この制約下での分解は二元配置 ANOVA の主効果・交互作用分離と同様に**一意に定まる**
- 損失: 観測マスク付き残差再構成誤差 + bias 制約
- **意図**: 提案の中核（中心化が担当範囲を固定する仕組み）を数式レベルで示す。「解釈可能」の根拠がここにあることを明確にする

### P10: 実験設定

- 2つの観点: (1) 真の成分が既知の合成データで成分回復を確認、(2) FreshRetailNet で補正が有効になる条件を確認
- FreshRetailNet: 店舗×商品系列、28日履歴 → 2日先×24時間帯、最大6000系列学習・5 seed
- 指標: cell MAE / WAPE / bias / high residual MAE（基準値誤差上位10%）/ hour corr（時間帯成分の解釈性）
- **意図**: 「精度」と「解釈性」の両方を測る評価設計であることを伝える。特に hour corr と high residual MAE は独自の観点なので一言添える

### P11: 実験結果 Synthetic（1）— 成分回復の主結果

- 表: base 条件で global corr 0.9995 / day 0.9979 / hour 0.9991 / interaction 0.9654
- interaction が強い条件では interaction corr 0.9890 まで上昇 → 必要な条件で役割が明確になる
- **意図**: 「成分が存在すれば提案モデルは正しく回復できる」という主張の根拠。表は主要行のみに絞る

### P12: 実験結果 Synthetic（2）— 回復の可視化

- 図（fig2）: subgroup ごとの真の時間帯成分 $c_h$ と推定成分 $\hat c_h$ の重ね描き + 推定成分のみの t-SNE で subgroup が分離
- **意図**: 数値だけでなく「成分がプロファイルとして読める」ことを視覚的に見せる。発表で一番映えるページ

### P13: 実験結果 Synthetic（3）— 成立条件と失敗条件

- small sample では interaction corr が 0.4252 まで低下 → 相互作用成分は系列数に敏感
- 中心化なしのモデルは残差 MAE は悪くなくても global corr が負になり成分が崩れる
- **予測精度と成分解釈性は別物**であり、中心化は正則化ではなく解釈のための構造化
- **意図**: 限界を正直に示して信頼性を上げる。「中心化がなぜ必要か」の実験的裏付けでもある

### P14: 実験結果 FreshRetailNet（1）— latent 分割 vs 成分別出力

- 表: 同一 residual target 上の統制比較。latent 4分割は global/local 型（0.0609）を上回らない。成分別出力+中心化は corrected MAE 0.0572、high residual 0.2656（baseline 0.2923）、hour corr 0.99+
- 補正性能・高残差改善・時間帯解釈の**すべてが成立したのは中心化付き成分別出力のみ**
- **意図**: 「latent を分けるだけでは足りない」ことを実データで示す。提案の中核を出力制約に置く根拠であり、発表の山場

### P15: 実験結果 FreshRetailNet（2）— 基準値の感度と統計的検証

- `series_mean_all`: baseline MAE 0.0697 → 補正後 0.0501、high residual 0.1914、hour corr 0.9938
- `same_hour_d7_all`: baseline 自体が強く（0.0580）、補正後 0.0565 と改善は小さい
- 5 seed の paired bootstrap: corrected cell MAE の改善 −0.0196（95% CI [−0.0201, −0.0189]）、high residual の改善 −0.0874（95% CI [−0.0905, −0.0842]）→ seed 依存の偶然ではない
- **意図**: 効く条件（`series_mean`）での改善が統計的に一貫していることを数値で示す。「1回の実験でたまたま出た結果では」という質疑を先に潰す

### P16: 実験結果 FreshRetailNet（3）— hour profile による残差診断

- 図（fig1）: 観測残差の時間帯平均 profile $\bar r_h$ とモデルの時間帯成分 $\bar c_h$ の重ね描き
- (a) 系列平均基準値: 残差に時間帯構造が残り、成分が corr 0.996 で対応。早朝は過大予測・朝夕ピークは過小予測という運用上の読みがそのまま得られる
- (b) 同時間帯直近平均基準値: 残差 profile の振幅が約1桁小さい → 基準値が時間帯構造を吸収済み
- 効かない条件は失敗ではなく、**基準値が何を吸収済みかを示す診断情報**として読める
- **意図**: 「常に勝つ」と主張しない誠実さと、「効く条件・効かない条件を診断できる」という強みへの転換を図1枚で見せる。FreshRetailNet パートの締め

### P17: 現時点の結論

- 主張: 提案法は強い baseline を置き換える手法ではなく、**baseline 後に残る構造を診断し、必要な場合に補正と解釈を与える手法**
- 確立したこと: (1) synthetic での成分回復、(2) 中心化の必要性、(3) 残差構造が残る target での補正・高残差改善
- 実務上の意味: 各成分は発注・補充・販促・人員配置の見直し単位に対応する診断器になる
- **意図**: 中間発表として「何がどこまで確立したか」を明確に区切る。主張の範囲を自分から限定することで質疑を制御する
