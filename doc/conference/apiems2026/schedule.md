## 2026-07-31 までの提出手順

国際会議原稿は、英語版から直接作り始めない。
まず日本語で論文の主張、数式、実験の読み方を固め、教授に提出する。

手順は次の通り。

```text
1. 日本語論文を書き上げて教授に提出する。
2. 日本語版の内容で OK をもらう。
3. OK が出た版を英語に翻訳する。
4. 英語版を教授に再確認してもらう。
5. 教授の OK 後、2026-07-31 までに投稿する。
```

英語版では、主提案を `residual correction`, `4変数への分解`, `centering constraints` として書く。
`global/local` は出発点として扱い、主張の中心は「baseline 後に残る residual を series/day/hour/interaction の出力補正量として分けること」に置く。
